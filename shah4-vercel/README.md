# shah4-site

Official player website for Elijah Shahid Muhammad featuring measurables, scouting profile, film links, and contact information.

## Deploy on Vercel

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to Vercel and import the GitHub repository.
4. Click **Deploy**.
5. In Vercel, open **Project Settings → Domains**.
6. Add your domain and follow the DNS instructions Vercel gives you.

## Run locally

```bash
npm install
npm run dev
```

## Replace next

- Update film links in `app/page.tsx`
